# Pokémon World App

Contents of App:
* Landing page - complete with spinning Pikachu!
* Pokédex page - 500 Pokémon with their stats
* Pokémon Quiz page - test your knowledge
* Poke the Pokémon - how many can you poke in 10 seconds?
